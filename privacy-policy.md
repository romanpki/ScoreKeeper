# Politique de confidentialité — ScoreKeeper
*Dernière mise à jour : mars 2026*

ScoreKeeper est une application personnelle de comptage de points pour jeux de cartes.

## Données collectées
L'application enregistre uniquement les données que vous saisissez vous-même : noms des joueurs et scores des parties. Aucune donnée personnelle sensible n'est collectée.

## Stockage des données
Les données sont stockées localement sur votre appareil via AsyncStorage. Si vous utilisez iCloud, elles sont synchronisées via iCloud Key-Value Storage dans votre propre compte Apple. Ces données ne transitent jamais par un serveur tiers.

## Partage des données
Aucune donnée n'est partagée avec des tiers, ni vendue, ni utilisée à des fins publicitaires ou analytiques.

## Suppression des données
Vous pouvez supprimer toutes vos données en désinstallant l'application ou en réinitialisant iCloud depuis les réglages de votre appareil.
